# Путеводитель по гиту

Команда смены директории
```sh
cd c:\folder_name
```

Команда отображения текущей директории для Mac & Linux
```sh
pwd
```

Листинг текущей директории

Win:
```sh
dir
```
Mac & Linux:
```sh
ls
```

Команда для удаления файла в текущей системе

Win:
```sh
del filename
```

Mac & Linux
```sh
rm filename
```
